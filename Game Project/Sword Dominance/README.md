# 칼의 지배 (Sword Dominance)

## 게임 소개
메탈슬러그 스타일의 횡스크롤 런앤건 게임입니다.

**배경**: 고려시대, 거란의 침략에 맞서는 무인의 이야기

**주인공**: 고려시대 말단 무인

## 게임 시스템

### 조작키
- **이동**: 방향키 (← →)
- **액션**: QWER 키
  - Q: 기본 공격
  - W: 특수 공격
  - E: 아이템/상호작용
  - R: 궁극기

### 게임 플레이
- 수평으로 스크롤되는 스테이지
- 적을 처치하며 진행
- 점진적으로 어려워지는 난이도

# 게임 설계 문서 (GDD)
## 칼의 지배 (Sword Dominance)

### 1. 게임 개요
- **장르**: 횡스크롤 런앤건 액션 게임
- **스타일**: 메탈슬러그 유사
- **배경**: 고려시대, 거란의 침략
- **플랫폼**: PC (Godot 4.2)

### 2. 핵심 게임플레이

#### 주인공
- 고려시대 말단 무인
- 체력: 100 HP (기본값)
- 움직임: 빠르고 민첩함

#### 조작 키
| 키 | 액션 | 설명 |
|---|---|---|
| ← → | 이동 | 좌우 이동 |
| Q | 공격 | 기본 칼 공격 |
| W | 특수 공격 | 범위 또는 강력한 공격 |
| E | 상호작용 | 아이템 획득, NPC 대화 |
| R | 궁극기 | 강력한 필살기 (쿨타임 있음) |
| Space | 점프 | 높이뛰기 (구현 예정) |

### 3. 게임 메커니즘

#### 전투 시스템
- 좌우 방향만 이동 가능
- 적과의 충돌 시 데미지
- 공격으로 적 제거
- 일부 적은 여러 번 공격 필요

#### 스테이지 구성
- 선형 진행 (왼쪽에서 오른쪽으로)
- 연속된 적 스폰
- 보스 스테이지 (설계 예정)
- 다음 스테이지로 진행 문 또는 플래그

#### 보상 시스템
- 적 처치 시 점수 획득
- 점수로 순위표 (하이스코어)
- 건강 회복 아이템 드롭 (가능성)

### 4. 아트 스타일
- **그래픽**: 픽셀 아트 (2D 스프라이트)
- **색감**: 역사적 한국 색감 (갈색, 금색, 빨강)
- **카메라**: 고정 또는 약간 따라다니는 캐메라

### 5. 음악 및 사운드
- **배경음악**: 한국 고대 음악 스타일
- **효과음**: 검 소리, 발걸음, 적 처치 음
- **음원**: 외부 제공 예정

### 6. 스테이지 구성 (예정)
1. **튜토리얼**: 기본 조작 학습
2. **초급 스테이지**: 기본 적 1-2종류
3. **중급 스테이지**: 다양한 적 타입
4. **최종 스테이지**: 거란 장수 보스 전
5. **엔딩**: 스토리 클로징
